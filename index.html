Huong dan chay nhanh

  1. Mo SQL Server Management Studio (SSMS).
  2. Mo file TableQuanLyChiTieuCaNhan_v2.sql.
  3. Chinh bien @ForceRecreate = 1 neu muon tao lai database tu dau.
  4. Execute script theo block (GO).
  5. Kiem tra du lieu bang truy van: SELECT TOP 10 * FROM GiaoDich;
